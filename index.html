Configuración de Firebase — solo primera vez

1
Entrá a console.firebase.google.com con tu cuenta Google y creá un proyecto (ej. faltantes-app).
2
Dentro del proyecto → click en ⚙️ Configuración del proyecto → sección "Tus apps" → ícono </> (Web) → registrá la app → copiá apiKey y projectId.
3
Menú izquierdo → Firestore Database → Crear base de datos → modo producción → región us-central1.
4
Firestore → pestaña Reglas → cambiá if false por if true → Publicar.
