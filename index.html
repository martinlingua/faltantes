// Configuración de Firebase (solo primera vez)

1
Entrá a console.firebase.google.com con tu cuenta Google y creá un proyecto nuevo (ej. faltantes-app). Podés desactivar Google Analytics.
2
Dentro del proyecto, hacé click en el ícono </> (Web), dale un nombre a la app y registrala. En la pantalla que sigue verás un objeto firebaseConfig — copiá el apiKey y el projectId.
3
En el menú izquierdo → Firestore Database → Crear base de datos → modo producción → región us-central1 → Listo.
4
En Firestore → pestaña Reglas, pegá esto y publicá:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
